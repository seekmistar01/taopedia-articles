Quasar is Bittensor Subnet 24. Its on-chain identity registers the subnet as built to push past the
long-context barrier in language models.

## What Quasar Provides

The registered identity describes Quasar as a "Bittensor subnet built to crush the long-context
barrier" — the problem of getting language models to use very long inputs effectively. Miners
compete to produce results against that objective, and the network rewards the strongest. The
registered project website is [silxinc.com](https://silxinc.com/) and the codebase is the
[SILX-LABS/QUASAR-SUBNET](https://github.com/SILX-LABS/QUASAR-SUBNET/) repository, which defines the
task in more detail than the on-chain identity alone.

## Miner and Validator Roles

Subnet 24 operates under the standard Bittensor two-role structure. Miners supply a capability to
the network, and validators evaluate those contributions and set weights. Reward distribution
follows [[Yuma Consensus]].

## On-Chain Identity

The live Finney identity for netuid 24 registers the subnet name as Quasar, with the description
"Bittensor subnet built to crush the long-context barrier." The registered project website is
[silxinc.com](https://silxinc.com/) and the GitHub repository is
[SILX-LABS/QUASAR-SUBNET](https://github.com/SILX-LABS/QUASAR-SUBNET/); a Discord channel is also
recorded in the identity. Live subnet data is available on
[TaoStats](https://taostats.io/subnets/24).
