# TAO Weight

TAO weight describes how TAO stake counts relative to alpha stake when Bittensor derives stake
weight inside a subnet
([Glossary: TAO Weight](https://docs.learnbittensor.org/resources/glossary#tao-weight),
[Glossary: Stake Weight](https://docs.learnbittensor.org/resources/glossary#stake-weight)).

The term is part of stake-weight vocabulary. It is not a separate token, staking role, or validator
score by itself.

## What It Represents

TAO weight is a protocol-level parameter that governs how TAO stake counts relative to alpha stake
when stake weight is formed. Stake weight combines alpha stake, TAO stake, and TAO weight; that
merged value scales validator influence in Yuma Consensus aggregation and validator permit ranking
([Glossary: Stake Weight](https://docs.learnbittensor.org/resources/glossary#stake-weight),
[Yuma Consensus](https://docs.learnbittensor.org/learn/yuma-consensus),
[Glossary: Validator Permit](https://docs.learnbittensor.org/resources/glossary#validator-permit)).

TAO-weight examples from one network environment should not be read as representing production
stake-weight conditions on another
([Bittensor Networks](https://docs.learnbittensor.org/concepts/bittensor-networks)).

## Distinction from Stake Weight

TAO weight names the relative role of TAO stake when stake weight is formed. Stake weight is the
computed influence value that combines alpha stake, TAO stake, and TAO weight inside a subnet
([Glossary: TAO Weight](https://docs.learnbittensor.org/resources/glossary#tao-weight),
[Glossary: Stake Weight](https://docs.learnbittensor.org/resources/glossary#stake-weight)).

- **TAO weight** — the protocol parameter that weights the TAO-stake component.
- **Stake weight** — the merged influence measure used in subnet consensus context.

## Distinction from Validator Weights

TAO weight helps determine stake-derived influence. Validator weights express a validator's miner
evaluations inside Yuma Consensus
([Glossary: TAO Weight](https://docs.learnbittensor.org/resources/glossary#tao-weight),
[Glossary: Validator Weights](https://docs.learnbittensor.org/resources/glossary#weight-vector),
[Yuma Consensus](https://docs.learnbittensor.org/learn/yuma-consensus)).

- **TAO weight** — how TAO stake counts when forming stake weight.
- **Validator weights** — the validator's submitted miner evaluations.

## Distinction from Root Proportion

TAO weight is the relative-influence parameter for TAO stake when stake weight is formed. Root
proportion is a subnet-level dividend-context measure of how much of a subnet's dividend split is
attributed to TAO staked through Root
([Glossary: TAO Weight](https://docs.learnbittensor.org/resources/glossary#tao-weight),
[Glossary: Root Proportion](https://docs.learnbittensor.org/resources/glossary#root-proportion)).

- **TAO weight** — protocol parameter weighting the TAO-stake component in stake weight.
- **Root proportion** — subnet-level split reading for root-routed TAO dividends.

## Distinction from Relative Stake Weight

TAO weight governs how TAO stake counts when stake weight is formed. Relative stake weight expresses
one validator's stake weight as a proportion of the validator set on the same subnet
([Glossary: TAO Weight](https://docs.learnbittensor.org/resources/glossary#tao-weight),
[Glossary: Relative Stake Weight](https://docs.learnbittensor.org/resources/glossary#relative-stake-weight)).

- **TAO weight** — parameter weighting the TAO-stake component in stake weight.
- **Relative stake weight** — proportional share of merged stake weight within the validator set.

## Distinction from Root Claims

Root claims handle the alpha dividends earned through Root Subnet staking. TAO weight is the global
multiplier that sets how much root (TAO) stake counts relative to a subnet's alpha stake when stake
weight is computed
([Root Claims](https://docs.learnbittensor.org/staking-and-delegation/root-claims),
[Glossary: TAO Weight](https://docs.learnbittensor.org/resources/glossary#tao-weight)).

- **Root claims** — the handling rule for alpha dividends earned through Root staking.
- **TAO weight** — the multiplier that scales how much root stake counts against alpha stake.

## Distinction from Yuma Consensus

TAO weight names stake-mix policy inside stake-weight formation. That global multiplier vocabulary
shapes how root TAO stake counts toward influence; it is not the tempo pass that credits emission
shares from aggregated validator weights
([Glossary: TAO Weight](https://docs.learnbittensor.org/resources/glossary#tao-weight),
[Glossary: Stake Weight](https://docs.learnbittensor.org/resources/glossary#stake-weight)).

Yuma Consensus reads validator weights, applies stake weighting, clipping, and bonding, and converts
the result into miner incentives and validator dividends on the selected netuid
([Yuma Consensus](https://docs.learnbittensor.org/learn/yuma-consensus),
[Emission](https://docs.learnbittensor.org/learn/emissions)).

TAO weight adjusts one part of stake-weight computation across the network. Validator weight
submissions still determine evaluation signals Yuma aggregates each tempo
([Glossary: Weight Vector](https://docs.learnbittensor.org/resources/glossary#weight-vector)).

- **TAO weight** — multiplier weighting root (TAO) stake in stake-weight calculation.
- **Yuma Consensus** — epoch-boundary settlement from validator weights to emission shares.

## Reader Boundary

TAO weight names the relative role of TAO stake inside stake-weight computation, not validator
weights, rank, trust, dividends, or a standalone influence score. It is a single global network
parameter that applies wherever stake weight is calculated
([Glossary: TAO Weight](https://docs.learnbittensor.org/resources/glossary#tao-weight),
[Glossary: Stake Weight](https://docs.learnbittensor.org/resources/glossary#stake-weight)).

## Distinction from Root Staker

TAO weight and a root staker relate as a parameter and a participant it affects. TAO weight is the
global coefficient setting how much root (TAO-denominated) stake counts relative to subnet alpha
stake in consensus, while a root staker is a participant who stakes on the root subnet and whose
influence that coefficient scales
([Glossary: TAO Weight](https://docs.learnbittensor.org/resources/glossary#tao-weight),
[Glossary: Root Subnet/Subnet Zero](https://docs.learnbittensor.org/resources/glossary#root-subnetsubnet-zero),
[Glossary: Root Proportion](https://docs.learnbittensor.org/resources/glossary#root-proportion)).

- **TAO weight** — the coefficient scaling root stake against alpha stake.
- **Root staker** — a participant staking on the root subnet, scaled by TAO weight.
