# NaCl Format

NaCl Format is Bittensor wallet-file terminology for a secure encryption format. The official
[NaCl Format glossary](https://docs.learnbittensor.org/resources/glossary#nacl-format) describes it
as a format that uses the NaCl library and is used when improving security for legacy Bittensor
wallets.

## Format Context

NaCl Format is not a Bittensor network role, subnet role, asset type, or address format. In Taopedia
context, the term is best read as encryption-format vocabulary for wallet-file protection.

## Wallet Context

Bittensor documentation treats wallets and keys as the surrounding context for wallet-file security.
NaCl Format sits beside that vocabulary as a file-protection term: it concerns the format used for
legacy wallet security, not a new wallet category.

## Wallet File, Seed Phrase, and Chain Identity

NaCl Format is useful when a reader needs to separate local wallet-file protection from the wallet's
on-chain identity. The wallet docs describe a Bittensor wallet through key pairs, balances, stake,
transactions, and subnet interactions; the NaCl glossary entry instead names a secure encryption
format used when updating legacy Bittensor wallets. In that sense, NaCl Format is about how
sensitive wallet material is protected locally, not about changing a wallet address, coldkey role,
hotkey role, or staking record.

It is also different from a seed phrase. A seed phrase is recovery material for a wallet's
cryptographic key pair, so losing it can mean losing access and leaking it can give someone else
control. NaCl Format may describe protected wallet-file storage, but it does not replace
recovery-phrase handling or make an exposed phrase safe.

For Taopedia readers, the short version is: use NaCl Format for the wallet-file security format, use
seed phrase or mnemonic for recovery material, and use wallet, coldkey, hotkey, or address when
talking about network identity and signing authority.

## Reading the Term

This article is a concept boundary. It does not provide wallet handling steps. For a reader, the
important point is that NaCl Format names an encryption format used in Bittensor wallet security
context.
