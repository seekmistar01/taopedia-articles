# Proposal Hash

A proposal hash is the identifier used to refer to a proposal in Bittensor governance. The
[Glossary](https://docs.learnbittensor.org/resources/glossary#proposal-hash) defines it as a unique
identifier for a proposal used in the voting process.

## Governance Context

The [Senate](https://docs.learnbittensor.org/governance/senate) documentation places proposals in
the Senate review process, while the
[Governance Overview](https://docs.learnbittensor.org/governance) explains that proposals are
created by the Triumvirate and reviewed by the Senate. A proposal hash is the reference for the
proposal, not the proposal body itself.

## Relationship to Proposal

A proposal hash and a proposal refer to related but different parts of Bittensor governance. The
[Glossary: Proposal](https://docs.learnbittensor.org/resources/glossary#proposal) describes a
proposal as a suggestion or plan put forward by the Triumvirate for the Senate to vote on. The
[Glossary: Proposal Hash](https://docs.learnbittensor.org/resources/glossary#proposal-hash)
describes the proposal hash as the unique identifier for a proposal used in the voting process.

For readers, the proposal is the governance item being reviewed, while the proposal hash is the
identifier used to refer to it during Senate voting. A proposal hash should not be read as the
proposal body itself, only as the reference that points to it.

References: [Glossary: Proposal](https://docs.learnbittensor.org/resources/glossary#proposal),
[Glossary: Proposal Hash](https://docs.learnbittensor.org/resources/glossary#proposal-hash)

## Reader Boundary

This article does not list active proposal hashes or describe any proposal outcome. Readers should
use official governance references for proposal records and review context.
