Claims is Bittensor Subnet 111. Its on-chain identity registers the subnet for turning scientific
literature into a structured claim-evidence graph.

## What Claims Provides

The registered identity describes Claims as a subnet "turning scientific literature into a
structured claim-evidence graph." Miners produce that structured output, and the network rewards the
strongest results. The codebase is the
[oneoneone-io/subnet-111](https://github.com/oneoneone-io/subnet-111) repository, which defines the
task in more detail than the on-chain identity alone.

## Miner and Validator Roles

Subnet 111 operates under the standard Bittensor two-role structure. Miners supply a capability —
here structured claim-evidence extraction — and validators evaluate those contributions and set
weights. Reward distribution follows [[Yuma Consensus]].

## On-Chain Identity

The live Finney identity for netuid 111 registers the subnet name as Claims, with the description
"Turning scientific literature into a structured claim-evidence graph." The GitHub repository is
[oneoneone-io/subnet-111](https://github.com/oneoneone-io/subnet-111); no project website URL or
Discord is set in the identity. Live subnet data is available on
[TaoStats](https://taostats.io/subnets/111).
