Chunking is Bittensor Subnet 40. Its live Finney identity registers the subnet name as "Chunking"
and lists a VectorChat GitHub path, but it does not currently provide a subnet URL, contact address,
Discord field, or public description.

The registered GitHub path did not return a loadable public repository or README when checked for
this article. Because Taopedia subnet articles should be grounded in current primary sources, this
page treats Subnet 40 as an identified subnet whose public implementation source is not presently
available through the live identity fields.

## Current Status

For active subnet articles, Taopedia normally describes what miners do, how validators score them,
and which repository or documentation defines the mechanism. Subnet 40's live identity confirms the
name Chunking, but the identity fields do not currently expose enough public source material to
support a reader-facing mechanism description.

Readers should therefore verify the live Finney identity before relying on any older external
description of the subnet. If the operator restores the registered repository or publishes updated
documentation, this page can be expanded with miner and validator roles sourced from that material.

## On-Chain Identity

Live SN40 data is available on [TaoStats](https://taostats.io/subnets/40). Readers can reproduce the
subnet's registered identity fields with the Bittensor CLI command documented in the
[btcli reference](https://docs.learnbittensor.org/btcli/#btcli-subnets-get-identity):
`btcli subnets get-identity --netuid 40 --network finney`.

That Finney identity reports subnet name "Chunking" and a GitHub field naming the
`VectorChat/chunking_subnet` path. The subnet URL, contact, Discord, description, logo, and
additional fields are blank.
