CliqueAI is Bittensor Subnet 83. Its on-chain identity describes CliqueAI as an AI-powered maximum
clique solver network, lists [cliqueai.toptensor.ai](https://cliqueai.toptensor.ai/) as the subnet
URL, and lists [toptensor/CliqueAI](https://github.com/toptensor/CliqueAI) as the subnet's GitHub
repository.

## What CliqueAI Rewards

The CliqueAI README describes the subnet as a network for solving maximum clique problems. In the
mechanism documentation, validators select graph problems from a database, classify them by graph
scale and time limit, and send selected problems to eligible miners.

Miners return candidate clique solutions for those graph problems. The mechanism documentation
describes a submitted solution as a set of vertices, then scores it on optimality and diversity.
Optimality rewards valid maximal cliques and compares a miner's clique size against the strongest
solutions returned for the same problem. Diversity gives extra credit to valid solutions that are
not identical to many other miners' submissions.

## Miner and Validator Roles

CliqueAI validators coordinate the challenge loop. The mechanism documentation describes four
stages: problem selection, miner selection, scoring, and weight setting. Validators choose a problem
type, retrieve a problem from the database, filter eligible miners, sample miners with a
difficulty-adjusted probability, evaluate the submitted solutions, and update miner weights.

Miners compete by producing strong clique solutions under the challenge constraints. Because the
score combines solution quality with diversity, the subnet rewards miners that find large valid
cliques while still leaving room for different solver approaches.

## On-Chain Identity

Live SN83 subnet data is available on [TaoStats](https://taostats.io/subnets/83). Readers can also
reproduce the chain identity fields with the Bittensor CLI command documented in the
[btcli reference](https://docs.learnbittensor.org/btcli/#btcli-subnets-get-identity):
`btcli subnets get-identity --netuid 83 --network finney`.

That Finney identity reports the subnet name as CliqueAI, the subnet URL as
https://cliqueai.toptensor.ai/, the description as "CliqueAI - AI-Powered Maximum Clique Solver
Network", and the GitHub repo as toptensor/CliqueAI.
