sundae_bar is Bittensor Subnet 121. Its on-chain identity describes the subnet as a generalist AI
agent designed to execute real business workflows end-to-end, lists
[sundaebar.ai](https://www.sundaebar.ai/) as the subnet URL, and lists
[sundae-bar/bittensor-subnet](https://github.com/sundae-bar/bittensor-subnet) as the subnet's GitHub
repository.

## What sundae_bar Provides

SN121 is designed as a competitive pipeline for improving one generalist commercial AI agent. The
public README describes sundae_bar as defining structured Generalist Challenges, miners as
submitting public generalist agents, and validators as benchmarking those agents with an Agent Eval
Test Suite. The leading agent version is then intended to be hosted through the sundae_bar platform
for business users.

The website describes sundae_bar as a marketplace for production-ready AI agents and reusable
skills. It also lists Crumble, an autonomous security review agent for AI-generated software, as an
agent built by sundae_bar and continuously improved through SN121. This ties the subnet's evaluation
work to a public product surface rather than only to internal benchmarking.

## Miner and Validator Roles

Miners are developers who submit fully open-source generalist agents. The README says submissions
are archived and available for inspection, reuse, and competitive iteration. This makes miner work
auditable and lets later submissions build on earlier agent designs.

Validators evaluate submitted agents using AETS specifications derived from Generalist Challenges.
Those specifications include datasets, expected targets, rubrics, metrics, and graders. The README
describes validators running evaluations across multiple seeds, configurations, and scenario
variations, then aggregating results into a consensus performance score. The top-performing agent
for an evaluation window receives the subnet's miner incentive signal, while validator weights feed
into [[Yuma Consensus]].
