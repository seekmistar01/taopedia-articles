# Public Subtensor

Public Subtensor refers to publicly reachable Bittensor chain access. Official Bittensor
documentation uses Subtensor to describe the blockchain layer that supports Bittensor network state.

References:
[Glossary: Public Subtensor](https://docs.learnbittensor.org/resources/glossary#public-subtensor),
[Subtensor](https://docs.learnbittensor.org/subtensor-nodes)

## Network Context

The term is useful when distinguishing public chain access from local development context. Public
Subtensor belongs to the live network side of that distinction, while local blockchain context is
private and development-oriented.

References:
[Glossary: Public Subtensor](https://docs.learnbittensor.org/resources/glossary#public-subtensor),
[Bittensor Networks](https://docs.learnbittensor.org/concepts/bittensor-networks)

## Reader Boundary

This page defines the term. It does not provide endpoint recommendations, connection steps, or
node-operation guidance. Readers should use current official Bittensor documentation when they need
operational details.
