Endure Network is Bittensor Subnet 30.

## What Endure Network Provides

The on-chain description identifies Endure Network as a risk intelligence network. No public GitHub
repository or technical documentation is currently linked from the Finney identity, so the specific
miner task, validator evaluation method, and reward mechanics cannot be described from live on-chain
sources alone.

## Miner and Validator Roles

Subnet 30 operates under the standard Bittensor two-role structure. Miners supply a capability to
the network, and validators evaluate those contributions and set weights. Those weights feed into
[[Yuma Consensus]], which converts validator assessments into emissions that reward the strongest
performers.
