Subnet 42 is a Bittensor subnet slot whose current on-chain identity is registered as "Unknown." The
live identity does not list a public GitHub repository, website, contact address, Discord,
description, logo, or additional project metadata.

Because the registered identity does not publish project materials, this page does not describe a

guide.

## On-Chain Identity

Live SN42 data is available on [TaoStats](https://taostats.io/subnets/42). Readers can reproduce the
subnet's registered identity fields with the Bittensor CLI command documented in the
[btcli reference](https://docs.learnbittensor.org/btcli/#btcli-subnets-get-identity):
`btcli subnets get-identity --netuid 42 --network finney`.

That Finney identity reports netuid 42, subnet name "Unknown", and blank values for GitHub, contact,
URL, Discord, description, logo, and additional metadata.
