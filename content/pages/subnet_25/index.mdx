Mainframe is Bittensor Subnet 25, a decentralized science subnet from Macrocosmos. The GitHub
repository listed in the subnet's Finney identity is
[macrocosm-os/mainframe](https://github.com/macrocosm-os/mainframe).

## What Mainframe Provides

The Mainframe README describes Subnet 25 as a decentralized science subnet for scientific compute.
Its current focus is life-sciences work, including protein molecular dynamics and protein-ligand
docking, while the broader goal is to support generalized scientific compute on Bittensor.

In Bittensor terms, Mainframe turns scientific computation into a competitive subnet task. Miners
contribute compute outputs for scientific jobs, validators evaluate those outputs, and validator
weights feed into [[Yuma Consensus]].

## Miner and Validator Roles

Miners provide scientific-compute results for the jobs active on the subnet. The README describes
protein-folding and docking workloads as current examples, with the subnet designed to adapt to
other life-sciences compute problems over time.

Validators run the evaluation side of the subnet. They check submitted outputs for the active
scientific task and set weights according to the subnet's incentive mechanism. Operators should
treat the repository as the source of truth for current setup details.

## On-Chain Identity

Live SN25 data is available on [TaoStats](https://taostats.io/subnets/25). Readers can also
reproduce the chain identity fields with the Bittensor CLI command documented in the
[btcli reference](https://docs.learnbittensor.org/btcli/#btcli-subnets-get-identity):
`btcli subnets get-identity --netuid 25 --network finney`.

That Finney identity reports the subnet name as Mainframe, the subnet URL as macrocosmos.ai/sn25,
and the GitHub repo as macrocosm-os/mainframe.
