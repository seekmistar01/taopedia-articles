# Mainchain

Mainchain is the glossary term for the primary Bittensor blockchain network. The
[Glossary](https://docs.learnbittensor.org/resources/glossary#mainchain) describes it as the primary
Bittensor network used for production purposes.

## Network Context

The [Bittensor Networks](https://docs.learnbittensor.org/concepts/bittensor-networks) reference
lists mainnet alongside testnet and localnet. In Taopedia, Mainchain is best read as the production
chain context, while testnet and localnet name non-production contexts for testing and development.

## Reader Boundary

This article defines a network context. It does not list connection details, endpoint status, local
environment details, or current network conditions. Readers should use official network references
for network-specific details.
