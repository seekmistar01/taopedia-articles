Subnet 76 is a Bittensor subnet slot whose current on-chain identity is registered as Byzantium. The
live identity does not currently present a public repository, subnet website, description, or other
project documentation.

Readers should therefore treat Subnet 76 as an identity-labeled slot rather than as an article with
a reproducible public codebase or documented miner and validator mechanism. If the subnet identity
is later updated, the live Finney identity should be checked before relying on this page.

## Current Status

For active subnet articles, Taopedia normally describes what the subnet provides, what miners do,
and how validators score them. Subnet 76 does not currently expose that information through its live
identity fields.

The useful published fact is the registered identity state: netuid 76 is named Byzantium, while its
public source and description fields are blank.

## On-Chain Identity

Live SN76 data is available on [TaoStats](https://taostats.io/subnets/76). Readers can reproduce the
subnet's registered identity fields with the Bittensor CLI command documented in the
[btcli reference](https://docs.learnbittensor.org/btcli/#btcli-subnets-get-identity):
`btcli subnets get-identity --netuid 76 --network finney`.

That Finney identity reports the subnet name as Byzantium. It does not report a GitHub source,
subnet URL, contact, description, logo URL, or additional metadata.
