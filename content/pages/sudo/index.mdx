# Sudo

Sudo is the Bittensor glossary term for privileged administrative authority. The
[Glossary](https://docs.learnbittensor.org/resources/glossary#sudo) describes it as a privileged key
for administrative actions that has been replaced by the governance protocol for enhanced security.

## Governance Context

The [Governance Overview](https://docs.learnbittensor.org/governance) explains that Bittensor
governance moves network management away from centralized control over time. In that model, the
Triumvirate creates proposals and the Senate reviews them, replacing single-authority administration
with proposal review.

## Reader Boundary

This article describes Sudo as a governance term. It does not identify any live holder, expose key
material, or describe privileged action execution. Readers should use official governance references
for current proposal-review context.
