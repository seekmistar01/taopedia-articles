OxMarkets is Bittensor Subnet 35, a liquidity-focused subnet whose registered identity describes it
as liquidity infrastructure for the 0xMarkets perpetuals exchange. The GitHub repository listed in
the subnet's Finney identity is
[General-Tao-Ventures/cartha-validator](https://github.com/General-Tao-Ventures/cartha-validator),
which the README describes as the official validator implementation for the Cartha subnet.

## What OxMarkets Provides

The Cartha Validator README describes a validator system that scores miners from their USDC
liquidity positions and publishes weights to the Bittensor network. In this design, miner scoring is
based on position data such as locked amount, lock duration, pool weights, and whether positions or
participants are still eligible for the current epoch.

For Bittensor readers, the important point is that Subnet 35 maps liquidity commitments into subnet
weights. Validators replay and verify position data, compute miner scores, normalize those scores
into weights, and submit those weights to [[Yuma Consensus]].

## Miner and Validator Roles

Miners participate through liquidity positions that the validator system can verify for a scoring
epoch. The README describes miner scores as aggregating eligible positions, while filtering out
expired pools and deregistered participants.

Validators run the scoring and publication side of the subnet. The README describes weekly epoch
handling, verifier-based miner queries, score calculation, weight normalization, and repeated weight
publication across Bittensor epochs. Operators should treat the repository as the source of truth
for current participant guidance.

## On-Chain Identity

Live SN35 data is available on [TaoStats](https://taostats.io/subnets/35). The live Finney identity
for netuid 35 registers the subnet name as OxMarkets, with the description "Liquidity-as-a-Service
subnet powering 0xMarkets - a permissionless perp DEX for FX, crypto, and commodities." The
registered project website is [0xmarkets.io](https://www.0xmarkets.io/) and the GitHub repository is
[General-Tao-Ventures/cartha-validator](https://github.com/General-Tao-Ventures/cartha-validator).
