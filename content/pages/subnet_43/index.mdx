Graphite is Bittensor Subnet 43, a decentralized network for solving graph optimization problems.
Its current focus is the Traveling Salesman Problem (TSP) — finding the shortest route that visits a
set of points once and returns to the start — a problem whose difficulty grows sharply with the
number of points. The subnet turns that computation into a rewarded service: validators pose
problems, miners return solutions, and the closest-to-optimal solutions earn the most. The
incentive-mechanism code is maintained in the
[GraphiteAI/Graphite-Subnet](https://github.com/GraphiteAI/Graphite-Subnet) repository.

## What the Subnet Produces

The subnet's output is solved routes for graph problems. Because the optimal answer to a hard TSP
instance is expensive to compute, there is room for miners to compete on both solution quality and
speed. Problems can be expressed as coordinate inputs for undirected graphs or as edge inputs for
directed graphs, so a range of routing and ordering problems can be mapped onto the same task.

The repository ships several reference solvers — including a nearest-neighbour heuristic, an exact
dynamic-programming method, a beam search, and a learning-based hybrid pointer network — so a new
miner can participate without writing a solver from scratch. Operators are encouraged to bring
better algorithms, since the reward mechanism is designed to pay out more to solvers that beat the
bundled ones.

## Miner and Validator Roles

Validators currently generate synthetic graph problems and send them to miners. Each miner solves
the problem with whatever algorithm it runs and returns a route, within a time limit. Validators
then evaluate the returned solutions.

Scoring is comparative rather than absolute. According to the repository, a miner's reward depends
on how its solution compares both to the best solution in the cohort for that problem and to a
benchmark greedy heuristic; solutions that fail to beat the benchmark are penalized. This rewards
miners for genuinely better routes instead of for merely returning something. The resulting
validator scores become the weights that feed into [[Yuma Consensus]], which converts them into the
incentive split across miners and validators.

## On-Chain Identity

Live SN43 data, including metagraph state and alpha token pool information, is available on
[TaoStats](https://taostats.io/subnets/43). The live Finney identity for netuid 43 registers the
subnet name as Graphite, and points to the
[GraphiteAI/Graphite-Subnet](https://github.com/GraphiteAI/Graphite-Subnet) repository, which is the
source of truth for the algorithms and reward mechanism described above.
