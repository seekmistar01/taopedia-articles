TalkHead is Bittensor Subnet 108, a subnet for talking-head generation — producing a photorealistic
animated face that speaks in sync with given input. Its on-chain description is "Talking Head
Generation," and the codebase for the subnet is the `talkheadai/talkhead-subnet` repository. The
subnet turns the task of building a good talking-head model into a continuous, measured competition.

## What TalkHead Rewards

Rather than submitting finished videos, miners submit their actual models, packaged as containers so
the network can run them itself. The subnet rewards models that generate the best talking-head
output on a shared set of test inputs, judged on both the quality of the result and how quickly it
is produced. Because every miner's model is run on the same challenge set under the same conditions,
the comparison is standardized rather than self-reported.

Scoring follows a winner-take-all policy: for a given evaluation, the highest-scoring model receives
the reward and the others receive nothing, which pushes miners to keep improving rather than settle
for an average result.

## Miner and Validator Roles

Miners are the model builders. A miner trains a talking-head model, publishes it as a container
image, and submits a reference to that image to the subnet so it can be evaluated. The miner does
not run the evaluation itself; it only supplies the model.

Validators coordinate the evaluation and translate its results into rewards. They collect the
miners' submissions and hand them to an executor service that runs each model in a sandboxed GPU
environment, feeds it the same standardized challenge inputs, captures the outputs, and measures how
well and how fast each model performs. Validators then read those scores and set weights on the
network accordingly, which is how reward flows through [[Yuma Consensus]]. At the article level the
split is straightforward: miners supply talking-head models, while validators have them run on a
common benchmark, score the results, and weight the network.

## On-Chain Identity

Live SN108 subnet data is available on [TaoStats](https://taostats.io/subnets/108), which identifies
the subnet as TalkHead. The live Finney identity for netuid 108 registers the subnet name as
TalkHead, with the description "Talking Head Generation." The GitHub repository is
[talkheadai/talkhead-subnet](https://github.com/talkheadai/talkhead-subnet).
