Beam is Bittensor Subnet 105. The live Finney identity lists its subnet name as Beam, its GitHub
source as the Beam-Network organization repositories page, its subnet URL as https://b1m.ai, and its
description as "Decentralized bandwidth. A global network. Powering the open internet."

The Beam README describes the project as an open coordination layer for bandwidth and
machine-to-machine data transfer. It says Beam aggregates distributed bandwidth, coordinates data
transfers across participants, and uses Proof-of-Bandwidth to measure delivery performance during
transfers.

## What Beam Provides

Beam provides a coordination layer for moving data between sources and destinations. The README
describes transfer patterns that include one-to-one movement, fan-out from one source to multiple
destinations, aggregation from multiple sources, and mesh-style transfers.

BeamCore is the coordination stack behind this workflow. The public BeamCore transparency README
identifies reference code for transfer assignment and PRISM scoring, including orchestrator
selection, chunk allocation, stale reassignment, score computation, and scheduled metrics refresh.

## Miner and Validator Roles

Beam's public materials describe miners as orchestrators and workers. Orchestrators coordinate
bandwidth work through BeamCore, while workers move data directly between source and destination
connectors and submit Proof-of-Bandwidth evidence.

Validators read BeamCore scores and PRISM inputs and set weights on Bittensor. This maps Beam's
bandwidth-transfer workflow onto the broader [[Mining and Validating]] pattern, with validator
weights ultimately flowing through [[Yuma Consensus]].

## On-Chain Identity

Readers can reproduce the live identity fields with the Bittensor CLI command documented in the
[btcli reference](https://docs.learnbittensor.org/btcli/#btcli-subnets-get-identity):
`btcli subnets get-identity --netuid 105 --network finney`.

That Finney identity reports the subnet name as Beam, the GitHub source as the Beam-Network
repositories page, the subnet URL as https://b1m.ai, and the description "Decentralized bandwidth. A
global network. Powering the open internet."
