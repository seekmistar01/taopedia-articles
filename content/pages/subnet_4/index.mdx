# Subnet 4: Targon

Targon is Bittensor Subnet 4. [TaoStats](https://taostats.io/subnets/4) lists the live on-chain
identity name as Targon and describes it as an "Incentivized Compute Marketplace powered by the
Targon Virtual Machine (TVM)." The subnet's official
[manifold-inc/targon](https://github.com/manifold-inc/targon) repository presents Targon as a
confidential decentralized AI cloud and a secure AI infrastructure platform built around
Confidential Compute and Protected PCIe technology.

## What Targon Provides

According to the [official repository](https://github.com/manifold-inc/targon), Targon focuses on
secure AI infrastructure for decentralized compute. The project README says the platform uses
Confidential Compute and Protected PCIe technology to provide a secure execution environment from
hardware to application layers.

The same source says Targon is designed for verifiable and trustworthy operation across the stack,
including hardware-enforced protection, secure boot, remote attestation, and isolated execution
environments for AI workloads.

## Miner and Validator Roles

Targon follows the standard Bittensor subnet structure. Miners supply compute capability to the
network, and validators evaluate miner performance and set weights. Those validator assessments feed
into [[Yuma Consensus]], which determines how subnet emissions are distributed.
