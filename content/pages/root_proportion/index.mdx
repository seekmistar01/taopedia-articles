# Root Proportion

Root proportion is a Bittensor tokenomics term for the share of a subnet's dividend context
attributed to TAO staked through the root subnet
([Glossary: Root Proportion](https://docs.learnbittensor.org/resources/glossary#root-proportion),
[Glossary: Root Subnet/Subnet Zero](https://docs.learnbittensor.org/resources/glossary#root-subnetsubnet-zero)).

The measure helps distinguish root-routed TAO stake from the subnet's own alpha-token side of the
staking and emission model.

## What It Measures

Root proportion compares root-routed TAO stake with a subnet's alpha-token issuance context. The
official glossary frames it as a proportion between stake on Root and the subnet alpha-token supply
context
([Glossary: Root Proportion](https://docs.learnbittensor.org/resources/glossary#root-proportion),
[Emission](https://docs.learnbittensor.org/learn/emissions)).

For readers, the important point is not the formula mechanics. Root proportion identifies how much
of a subnet's dividend context belongs to root stakers rather than alpha-side stakers.

## Dividend Context

Root proportion is described on a zero-to-one range. In that framing, the value represents the
proportion of dividends attributed to root stakers for the subnet context being discussed
([Glossary: Root Proportion](https://docs.learnbittensor.org/resources/glossary#root-proportion),
[Emission](https://docs.learnbittensor.org/learn/emissions)).

This places root proportion inside Bittensor's wider emissions model. It is a supply-and-dividend
term, not a general subnet ranking or a statement about all subnet outcomes.

## Relationship to Root Subnet

Root proportion and Root Subnet describe related but different ideas. The Root Subnet is the staking
environment where TAO holders can stake in a subnet-agnostic way, while root proportion is the
subnet-level measure of how that root-routed stake shares in a subnet's dividend context
([Glossary: Root Subnet/Subnet Zero](https://docs.learnbittensor.org/resources/glossary#root-subnetsubnet-zero),
[Glossary: Root Proportion](https://docs.learnbittensor.org/resources/glossary#root-proportion)).

That distinction keeps the article vocabulary precise. The root subnet is a place in the protocol
model; root proportion is a measurement used when discussing a specific subnet's distribution
context.

## Relationship to TAO Weight

Root proportion is related to TAO weight, but the two terms answer different questions. TAO weight
is the global parameter that describes how TAO stake contributes relative to alpha stake, while root
proportion describes a subnet-level dividend split
([Glossary: TAO Weight](https://docs.learnbittensor.org/resources/glossary#tao-weight),
[Glossary: Root Proportion](https://docs.learnbittensor.org/resources/glossary#root-proportion)).

This distinction helps prevent root proportion from being treated as validator stake weight, TAO
weight, or total subnet emissions. It is narrower than each of those terms.

## Reader Boundary

A root-proportion claim should name the subnet and the dividend context being measured. It should
not be read as a staking recommendation, a quality score, or a claim that every subnet has the same
root-side distribution
([Glossary: Root Proportion](https://docs.learnbittensor.org/resources/glossary#root-proportion),
[Emission](https://docs.learnbittensor.org/learn/emissions)).

The stable article-level point is that root proportion describes root-routed TAO's share in a
subnet's dividend context.
