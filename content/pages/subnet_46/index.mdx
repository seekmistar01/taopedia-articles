Zipcode is Bittensor Subnet 46. Its on-chain identity describes the subnet as the Real Estate
Intelligence Network, lists [zipcode.ai](https://zipcode.ai/) as the subnet URL, and lists
[resi-labs-ai/RESI-models](https://github.com/resi-labs-ai/RESI-models) as the subnet's GitHub
repository.

## What Zipcode Provides

The RESI README describes Subnet 46 as a real estate price-prediction subnet. Miners compete by
building models for predicting United States residential property prices, while validators evaluate
the predictions against ground-truth sales data.

At the subnet level, Zipcode turns real-estate model accuracy into a miner-scoring signal. The
repository describes an evaluation design that separates model commitment from later sales-data
evaluation, so submitted models are tested on property data that was not available when the model
was committed.

## Miner and Validator Roles

Miners train and submit property price-prediction models. The README describes the public model
format as ONNX and frames the task around predicting residential property sale prices from property
features.

Validators fetch submitted models, run inference, calculate prediction scores, rank models, and set
weights on chain. The repository describes scoring with Mean Absolute Percentage Error, so lower
prediction error produces a stronger score. Operators should treat the repository as the source of
truth for current participant guidance.

## On-Chain Identity

Live SN46 data is available on [TaoStats](https://taostats.io/subnets/46). The live Finney identity
for netuid 46 registers the subnet name as Zipcode, with the description "The Real Estate
Intelligence Network." The GitHub repository is
[resi-labs-ai/RESI-models](https://github.com/resi-labs-ai/RESI-models).
