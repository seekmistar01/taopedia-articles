BitMind is Bittensor Subnet 34, also described by its repository as GAS, the Generative Adversarial
Subnet. The GitHub repository listed in the subnet's Finney identity is
[BitMind-AI/bitmind-subnet](https://github.com/BitMind-AI/bitmind-subnet).

## What BitMind Provides

The BitMind README describes GAS as an adversarial subnet for synthetic media. The subnet has two
parallel competition tracks: discriminative mining, where miners submit detection models for
AI-generated content, and generative mining, where miners return synthetic media for validator
challenges.

This design makes the subnet a Bittensor-native competition between detection and generation.
Detection miners improve models that identify synthetic image, video, and audio content, while
generation miners try to produce convincing synthetic outputs that pass validation and challenge the
detectors. Validator weights from those evaluations feed into [[Yuma Consensus]].

## Miner and Validator Roles

Discriminative miners submit detection models for evaluation across the supported media modalities.
The README describes these models as being benchmarked by validators, with rewards tied to detection
accuracy and calibration.

Generative miners run services that create synthetic media in response to validator prompts.
Validators check those outputs, evaluate detection models against real and synthetic media, and set
weights according to the subnet's incentive mechanism. Operators should treat the repository as the
source of truth for current participant guidance.
