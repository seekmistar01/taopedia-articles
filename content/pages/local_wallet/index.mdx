# Local Wallet

A local wallet is a Bittensor wallet whose files are kept on a user's machine. The
[Local Wallet glossary entry](https://docs.learnbittensor.org/resources/glossary#local-wallet)
places the term in wallet-location vocabulary rather than subnet, consensus, or tokenomics
vocabulary.

## Location Context

Local wallet names where wallet files live. The
[wallet documentation](https://docs.learnbittensor.org/keys/wallets) provides the broader Bittensor
wallet context, while the local-wallet glossary entry narrows the term to wallet files associated
with the user's machine.

## Wallet Location Relationship

Local wallet and wallet location are related but not identical. A
[wallet location](https://docs.learnbittensor.org/resources/glossary#wallet-location) is the
directory path where generated wallet files are kept, while local wallet names the wallet context
that uses that local storage model. One term points to the path; the other names the wallet
category.

## External Wallet Contrast

The contrast term is external wallet. An
[external wallet](https://docs.learnbittensor.org/resources/glossary#external-wallet) refers to
wallet software or custody outside the local Bittensor wallet path. Local wallet is the opposite
side of that distinction: wallet files are associated with the user's own local environment.

## Wallet-Key Context

Local wallet sits near wallet-key vocabulary because wallet files are part of how Bittensor users
organize access. The [wallet documentation](https://docs.learnbittensor.org/keys/wallets) supplies
that broader setting. The term local wallet does not rename public keys, private keys, or wallet
addresses; it only describes local wallet-file context.

## Address-Safety Boundary

Local wallet does not remove destination-address risk. The
[address-poisoning guide](https://docs.learnbittensor.org/keys/address-poisoning-scams) concerns
lookalike addresses during wallet use, while local wallet describes where wallet files are kept.
Those topics can meet during ordinary wallet activity, but they answer different questions.

## Reading the Term

Local wallet should be read as wallet-location vocabulary. It does not name a subnet role, validator
recommendation, balance state, or recovery process. The
[Local Wallet glossary entry](https://docs.learnbittensor.org/resources/glossary#local-wallet)
anchors the concept, with wallet-location and external-wallet references supplying nearby context.
