# Root Proportion

Root proportion is a Bittensor tokenomics term for the portion of a subnet's dividend context
attributed to TAO staked through the root subnet
([Glossary: Root Proportion](https://docs.learnbittensor.org/resources/glossary#root-proportion),
[Glossary: Root Subnet/Subnet Zero](https://docs.learnbittensor.org/resources/glossary#root-subnetsubnet-zero)).

The measure helps distinguish root-routed TAO stake from the subnet's own alpha-token side of the
staking and emission model.

## Dividend Context

Root proportion is described on a zero-to-one range. In that framing, the value represents the
portion of subnet dividends attributed to root stakers for the subnet context being discussed
([Glossary: Root Proportion](https://docs.learnbittensor.org/resources/glossary#root-proportion),
[Emission](https://docs.learnbittensor.org/learn/emissions)).

This places root proportion inside Bittensor's wider emissions model. It is a supply-and-dividend
term, not a general subnet ranking or a statement about every subnet outcome.

## Root Subnet Relationship

Root proportion and Root Subnet describe related but different ideas. The Root Subnet is the staking
environment where TAO holders can stake in a subnet-agnostic way, while root proportion is the
subnet-level measure of how that root-routed stake participates in a subnet's dividend context
([Glossary: Root Subnet/Subnet Zero](https://docs.learnbittensor.org/resources/glossary#root-subnetsubnet-zero),
[Glossary: Root Proportion](https://docs.learnbittensor.org/resources/glossary#root-proportion)).

That distinction keeps the vocabulary precise. Root Subnet names a protocol environment; root
proportion names a subnet-level dividend measure.

## TAO Weight Boundary

Root proportion is related to TAO weight, but the two terms answer different questions. TAO weight
describes how TAO stake contributes relative to alpha stake in stake-weight interpretation, while
root proportion describes a subnet-level dividend context
([Glossary: TAO Weight](https://docs.learnbittensor.org/resources/glossary#tao-weight),
[Glossary: Root Proportion](https://docs.learnbittensor.org/resources/glossary#root-proportion)).

This distinction helps prevent root proportion from being treated as stake weight, TAO weight, or
total subnet emissions. It is narrower than each of those terms.

## Stake-Weight Boundary

Stake weight and root proportion both sit near staking vocabulary, but they belong to different
parts of Bittensor. Stake weight is a validator-influence measure in subnet consensus, while root
proportion is a dividend-context measure for root-routed TAO
([Glossary: Stake Weight](https://docs.learnbittensor.org/resources/glossary#stake-weight),
[Glossary: Root Proportion](https://docs.learnbittensor.org/resources/glossary#root-proportion)).

This keeps consensus influence separate from dividend attribution. A validator's stake weight does
not replace the subnet-level root-proportion concept.

## Alpha-Side Boundary

Root proportion is useful because Bittensor subnet tokenomics can involve both root-routed TAO and
subnet alpha context. The term identifies the root-routed side of that dividend discussion rather
than the alpha-token side
([Glossary: Root Proportion](https://docs.learnbittensor.org/resources/glossary#root-proportion),
[Emission](https://docs.learnbittensor.org/learn/emissions)).

For readers, this keeps token vocabulary from collapsing into one generic staking label. Root
proportion is specifically about root-routed TAO in subnet dividend context.

## Reader Boundary

A root-proportion claim should name the subnet and dividend context being measured. It should not be
read as a staking recommendation, a quality score, or a claim that every subnet has the same
root-side distribution
([Glossary: Root Proportion](https://docs.learnbittensor.org/resources/glossary#root-proportion),
[Emission](https://docs.learnbittensor.org/learn/emissions)).

The stable article-level point is that root proportion describes root-routed TAO's portion of a
subnet's dividend context.
